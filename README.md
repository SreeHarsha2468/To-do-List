# Simple To-Do List API

A simple backend API for a To-Do List application built with **Node.js**, **Express**, and **MongoDB**.

---

## Features

- User registration with hashed passwords
- User login
- Create, read, update, and delete tasks
- Retrieve all tasks for a specific user
- Simple RESTful API structure

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
